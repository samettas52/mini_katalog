# Mini Katalog

## Proje Açıklaması
Bu proje, temel Flutter widget'larını (Row, Column, Container, GridView) ve sayfa yönlendirmelerini (Navigator) kullanarak geliştirilmiş bir katalog ve sepet simülasyonudur. Uygulama içerisinde statik JSON verileri ile dinamik ürün listeleme, ürün detay sayfası gösterimi ve temel durum yönetimi ile sepet işlemleri bulunmaktadır.

**Geliştirici:** Samet Taş

## Kullanılan Flutter Sürümü
- Flutter 3.44.2 (Stable)
